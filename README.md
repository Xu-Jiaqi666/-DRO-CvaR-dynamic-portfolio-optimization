## 📌 项目简介

面向 A 股动态投资组合优化问题，构建 **“情景感知 → 收益预测 → 分布鲁棒优化 → 动态调仓”** 完整框架。  
融合多源协变量：个股交易特征（隔夜收益、滞后收益）、市场基准（沪深300）、宏观变量（汇率、国债收益率）、政策指数、消费者信心、货币政策（M2、M1‑M2剪刀差）。  
核心贡献：将 **分布鲁棒优化（DRO）** 与 **CVaR 尾部风险控制** 结合，通过对偶 SDP 转化实现每日动态权重求解，并引入 SHAP 进行预测层可解释性分析。

---

## 🔧 核心方法

| 模块 | 方法 |
| :--- | :--- |
| **收益预测** | LightGBM（基准）、OLS、历史均值（对照） |
| **分布鲁棒优化** | 矩约束分布不确定集合 + CVaR 约束 |
| **对偶转化** | 拉格朗日对偶 → 半正定规划（SDP） |
| **求解器** | CVXPY + SCS |
| **可解释性** | SHAP（预测层）、特征重要性 |

## 📊 主要结果（基准设定）

| 指标 | 鲁棒优化组合 | 均权组合（基准） |
| :--- | :--- | :--- |
| 年化收益率 | **17.55%** | 8.98% |
| 年化波动率 | 22.95% | 19.21% |
| 夏普比率 | **0.677** | 0.364 |
| 最大回撤 | -24.91% | -20.75% |
| 卡玛比率 | 0.704 | 0.433 |

> 扣除 0.2%/0.3% 交易成本后，策略仍保持相对优势。详细回测曲线见 `results/` 文件夹。
